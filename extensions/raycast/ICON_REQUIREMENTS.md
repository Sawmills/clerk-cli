# Icon Requirements

The Raycast extension requires an `icon.png` file in the root directory.

## Specifications
- **Format**: PNG with transparency
- **Size**: 512x512 pixels
- **File name**: `icon.png`

## Options

### Option 1: Use Clerk Logo
Download the official Clerk logo from [Clerk's brand assets](https://clerk.com) and resize to 512x512px.

### Option 2: Create Custom Icon
Create a custom icon that represents Clerk admin functionality. Consider using:
- A key symbol (for authentication)
- A user/organization icon
- Clerk's brand colors

### Option 3: Use Generic Admin Icon
Use a generic admin panel or dashboard icon from icon libraries like:
- [SF Symbols](https://developer.apple.com/sf-symbols/) (macOS)
- [Heroicons](https://heroicons.com/)
- [Feather Icons](https://feathericons.com/)

## Temporary Workaround

For development, you can use any 512x512 PNG file as a placeholder. The extension will work without a proper icon, but it won't look professional in the Raycast store.
